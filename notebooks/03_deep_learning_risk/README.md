# Deep Learning Risk Notebooks

This folder contains hybrid deep-learning experiments for MASI downside-risk forecasting.

## Main Notebook

`01_lstm_var_es_hybrid_risk_modeling.ipynb`

This notebook:

- studies multivariate predictors for risk forecasting;
- trains the LSTM VaR model;
- estimates Expected Shortfall with a Ridge second stage;
- evaluates out-of-sample forecasts;
- adds regime-aware diagnostics and economic evaluation.
